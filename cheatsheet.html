element list


  • <h1> </h1> →

    H1 header

  • <h2> </h2> →

    H2 header

  • <h3> </h3> →

    H3 header

  • <h4> </h4> →

    H4 header

  • <h5> </h5> →
    H5 header
  • <h6> </h6> →
    H6 header
  • <b> </b> → Bold
  • <i> </i> → Italic
  • <strong> </strong> → Semantic bold
  • <em> </em> → Semantic italic
  • <mark> </mark> → Highlighted text. CSS can be used to change the color.
  • <sup> </sup> → Superscript
  • <sub> </sub> → Subscript
  • <abbr title="Abbreviation with tooltip">AWT</abbr> → AWT
  • <blockquote> </blockquote> →
    Quoted text
  • <code> </code> → Inline code
  • <pre> </pre> → 
    Preformatted block of code
  • <div> </div> →
    Generic block container
  • <span> </span> → Generic inline container
  • <header> </header> →
    Semantic layout tag for a header
  • <footer> </footer> →
    Semantic layout tag for a footer
  • <main> </main> →
    Semantic layout tag for the main part of a site
  • <section> </section> →
    Semantic layout tag for a section of the site
  • <article> </article> →
    Semantic layout tag for an article
  • <nav> </nav> →
    Semantic layout tag for navigation, useful with links
  • <ul> </ul> →
      Unordered list (bullet points)
  • <ol> </ol> →
      Ordered list (numbered points)
  • <li> </li> → List item, requires to be placed in <ul> or <ol> for functionality
  • <dl> </dl> →
    Description list, container that wraps the description list
  • <dt> </dt> →
    Description term, the item or keywords being described
  • <dd> </dd> →
    Description details, the explanation or definition of the term
  • <a href="https://google.com"> → Hyperlink (this leads to google)
  • <img src="assets/Image..png" alt="This is an image"> </img> →

    Sets an image. Src is the location of the file, alt is the text that is displayed for accessibility.
  • <audio src="assets/audiofile.mp3" controls> </audio> →

    Sets an audio file.
  • <video src="assets/0530.mp4" width="50%" height=auto controls> </video> →

    Sets a video file.
  • <iframe...> </iframe> →

    Allows embeds. This is a YouTube example.
  • <iframe...> </iframe> →

    Allows embeds. This is a Spotify example.
  • <form> </form> →
    Collects user input. Optionally uses <input>, <label>, <textarea>, <select> <option> or <button>.
  • <br> →
    Line break
  • <hr> →
    Horizontal rule
  • <meta> → Metadata for the HTML page, like character encoding, controlling viewport on mobile devices, describing the pages for search results, author information, auto-refresh, redirect people and controls social media previews.
  • <link> → Connects the page to external resources, most commonly for loading stylesheets (CSS), icons or fonts.
  • <script> → Allows JS (JavaScript) to be used.
  • <style> → Allows CSS (Cascading Style Sheets) to be used.
  • <title> → The page title. You can also see it on the page's tab!
